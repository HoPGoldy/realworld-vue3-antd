# realworld-vue3-antd

realworld 项目，基于 vue3 / typescript / antd 开发，在线预览地址：[realworld-vue3-antd](https://hopgoldy.github.io/realworld-vue3-antd/)。

## 启动

使用: `node 12.16.1+`, `yarn`

```bash
# 安装依赖
yarn

# 启动本地开发服务
yarn dev

# 打包
yarn build
```
